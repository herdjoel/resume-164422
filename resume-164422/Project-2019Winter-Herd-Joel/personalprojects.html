Scroll Box about Closures!

Benefit of Closures: Because of Lexical Scoping, if we encapsulate functions within functions and return those inner functions, we can declare functions that retain information, and also optimize the code so that the inner functions are not searching the outer (or worse, global) scope for variables. Instead, they will have the information they need to perform the specific operation we would like them to, with the efficiency of containing this information in the immediate outer scope, which reduces the time it takes to search for that information.
